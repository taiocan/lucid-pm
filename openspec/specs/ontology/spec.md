# Ontology Specification

> Generated by openlore v1.0.0 on 2026-06-07 07:57

## Purpose

Provides AI-driven suggestions for ontology modifications based on a data snapshot and vocabulary
constraints by querying an external generative AI service.

## Requirements

> `modules/ontology_suggest/src/suggester.rs`

### Requirement: SuggestProposals

The system SHALL generates ontology proposals by querying an external ai service (gemini) with a data snapshot and optional vocabulary constraints. if no constraints are provided, hardcoded defaults are used.

#### Scenario: SuccessfulSuggestionWithCustomConstraints
<!-- openlore-test: tags=smoke (auto) -->
- **GIVEN** A valid data snapshot and specific vocabulary constraints are provided
- **WHEN** The suggest_proposals operation is invoked
- **THEN** An HTTP request containing the snapshot and constraints SHALL be sent to the Gemini API
- **AND** A list of parsed ontology proposals from the API response SHALL be returned

#### Scenario: SuccessfulSuggestionWithDefaultConstraints
<!-- openlore-test: tags=smoke (auto) -->
- **GIVEN** A valid data snapshot is provided, but no vocabulary constraints are specified
- **WHEN** The suggest_proposals operation is invoked
- **THEN** Hardcoded default vocabulary constraints SHALL be used
- **AND** An HTTP request containing the snapshot and default constraints SHALL be sent to the Gemini API
- **AND** A list of parsed ontology proposals from the API response SHALL be returned

#### Scenario: ExternalAPIFailure
<!-- openlore-test: tags=regression (auto) -->
- **GIVEN** The external Gemini API is unavailable or returns an error status code
- **WHEN** The suggest_proposals operation is invoked
- **THEN** An error indicating the API failure SHALL be returned

#### Scenario: InvalidAPIResponseFormat
<!-- openlore-test: tags=regression (auto) -->
- **GIVEN** The external Gemini API returns a response that cannot be parsed into the expected JSON structure
- **WHEN** The suggest_proposals operation is invoked
- **THEN** An error indicating the response parsing failure SHALL be returned
